## 源码如下：

```c
#include<stdio.h>
int main()
{
	//打印九九乘法表
	printf("九九乘法表：\n"); 
	int x, y; //初始化打印的两个方向
	for (y = 1; y < 10; y++) //两层循环嵌套打印输出
	{
		for (x = 1; x <= y; x++)
		{
			printf("%d*%d=%2d ", y, x, x * y); //%2d表示固定输出两位
		}
		printf("\n"); //打印到行尾，换行
	}
	printf("This is the end.\n");
}
```
